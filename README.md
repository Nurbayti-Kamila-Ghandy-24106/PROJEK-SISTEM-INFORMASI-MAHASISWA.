Deskripsi Projek
Sistem Informasi Mahasiswa Sederhana adalah sebuah aplikasi yang digunakan untuk mengelola data mahasiswa seperti menambah, melihat, mengubah, dan menghapus data. Projek ini dibuat untuk membantu memahami konsep dasar pemrograman dan pengelolaan data secara terstruktur.

Yang Dibutuhkan Oleh Projek :
- Python
- Text editor atau IDE (VS Code, PyCharm, dll)
- Pengetahuan dasar Python

How to Use / Bagaimana Menggunakan Projek :
- Jalankan file program menggunakan Python
- Pilih menu yang tersedia untuk mengelola data mahasiswa.
- Masukkan data sesuai perintah pada program.
- Data mahasiswa akan ditampilkan atau diperbarui sesuai pilihan menu.
